## RNA-seq Analysis

This directory contains the computational workflows used for the RNA-seq analysis of human neutrophils.

## Overview 

The analysis investigates transcriptional changes associated with short-term antibiotic-induced disruption of the gut microbiota.

Samples correspond to neutrophils collected from healthy volunteers before and after antibiotic treatment, with or without LPS stimulation.


## Workflow Overview

The RNA-seq workflow includes the following steps:

1. Raw read processing and transcript quantification
2. Removal of unwanted technical variation
3. Differential gene expression analysis
4. Functional enrichment analysis
5. Visualization and summary of results

### Pipeline

Raw sequencing reads were processed using the **nf-core/rnaseq pipeline**.

Key steps include:

- read quality control
- alignment to the human genome
- transcript quantification using Salmon
- generation of gene-level count matrices

Reference genome: **GRCh38**

The pipeline was executed using Nextflow with Docker containers.

See ```run_nextflow.sh``` for execution command and ```params.yaml``` for pipeline parameters.

### Downstream Analysis (R)

Downstream statistical analyses were performed in R.

#### Input Data

The analysis requires:
- gene-level count matrix produced by the pipeline
- sample metadata
- genome annotation (GENCODE)

Example files:
- salmon.merged.gene_counts.tsv
- samplesheet.tsv
- gencode.v49.primary_assembly.annotation.gtf

The workflow includes:

#### 1. Removal of unwanted variation

Technical variation was removed using the **RUVSeq** framework.

This step generates normalized counts and RUV factors used in the differential expression analysis.

Script: ```scripts/01_remove_unwanted_variation.R```

#### 2. Differential gene expression analysis

Differential expression was performed using **DESeq2** with a paired design.

The design model accounts for:
- subject pairing
- experimental condition
- unwanted variation factors

Script: ```scripts/02_differential_expression.R```

### 3. Visualization and functional analysis

Visualization and functional analyses include:

- quality control plots
- PCA and MDS
- volcano plots
- DEG counts
- Venn diagrams
- Gene Ontology enrichment
- Reactome pathway enrichment
- DEG clustering
- transcription factor enrichment

Script: ```scripts/03_deg_visualization_and_enrichment.R``` and ```scripts/04_GSEA_heatmap.R```

The full analysis workflow is available in scripts/* and notebooks/RNAseq_analysis.Rmd

## Reproducibility

The computational environment can be reproduced using ```sessionInfo()``` included at the end of the scripts.

## Data availability

Raw sequencing data are available in NCBI BioProject: PRJNA1402444

Intermediate objects are not included in the repository but can be generated by running the analysis scripts.
